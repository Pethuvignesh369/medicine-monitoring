# Veterinary Medicine Monitoring

A web app to monitor medicine availability, utilization, and weekly requirements for veterinary facilities in Tamil Nadu, built for the Veterinary HealthTech Challenge.

## Features
- Add and manage facilities (Hospitals, Dispensaries, etc.).
- Track medicine stock, weekly requirements, and expiry dates.
- Log daily usage with real-time stock updates.
- Filter by facility type and export data (PDF/Excel).
- Responsive design with elegant navbar and toast notifications.
